# HPC Project – Matrix Operations Library

This repository is the starting point of a long‑term project to build a **high‑performance C++ matrix library** suitable for scientific computing and HPC (High Performance Computing) environments.  

The project evolves step by step:
- **Stage 1:** Core matrix operations (addition, multiplication, transpose)
- **Stage 2:** Advanced operations (determinant, inverse, identity)
- **Stage 3:** Benchmarking and performance measurement
- **Stage 4:** HPC optimization with Eigen, BLAS, OpenMP, MPI
- **Stage 5:** Containerization, reproducibility, and publishing

---

## 🎯 Project Goals
- Provide a clean, extensible C++ matrix library
- Bridge the gap between **learning projects** and **production‑ready HPC code**
- Demonstrate performance awareness and scalability
- Serve as a foundation for future research and scientific workloads
